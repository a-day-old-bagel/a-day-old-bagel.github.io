Demos

These WebGL demos have been tested on Firefox and Chrome in Linux and Windows only. Some of them have texture rendering issues on macOS and iOS, but I don't own any apple hardware with which to test them. I haven't implemented any loading screens, so you may have to wait a few seconds for some of the textures to load.

WebGL Billiards

Instructions: Left-click-drag (or double-touch-drag) to shoot. Right-click-drag (or touch-drag) to orbit, scroll (or pinch) to zoom. Zoomed view re-focuses on cue ball every time balls come to a stop.

WebGL Realtime Raytracer

WebGL Planet Shader with Animated Weather

Click-drag to rotate view. Scroll to zoom. Or use arrow keys to pan/zoom.

Spacebar pauses/unpauses the animation. Use [ and ] to decrease or increase

animation speed.
